# 涂图短视频 TuSDK Video 产品介绍

涂图·短视频 SDK 服务是涂图推出的一款适用于 Android 平台的短视频 SDK，提供了包括录制、编辑等基础功能以及美颜、滤镜、动态贴纸、场景特效、魔法效果、文字特效、微整形等特效。
